# 基于 NexDynIPC 的可微物理实现方案

## 1. 概述

### 1.1 核心目标

实现**可微接触动力学（Differentiable Contact Dynamics）**，为强化学习和机器人仿真提供平滑、连续的物理梯度。

### 1.2 技术挑战

| 挑战 | 传统方法 | 可微物理方案 |
|------|----------|--------------|
| 接触力梯度 | 不可导/阶跃状 | 平滑连续 |
| 碰撞检测 | 离散事件 | 连续距离场 |
| 梯度质量 | 噪声大 | 高质量梯度流 |
| 收敛性 | 神经网络难收敛 | 稳定收敛 |

### 1.3 核心创新

使用**三次 SDF（Signed Distance Function）**构建平滑的接触模型：
- 平滑的距离场：$\phi(\mathbf{x})$
- 连续的法线场：$\nabla \phi(\mathbf{x})$
- 连续的黑塞矩阵：$\nabla^2 \phi(\mathbf{x})$

---

## 2. 理论基础

### 2.1 符号距离函数（SDF）

**定义**：
```
φ(x) =  min ||x - y||   如果 x 在物体外部
       -min ||x - y||   如果 x 在物体内部
       y∈∂Ω
```

**性质**：
- $|\nabla \phi| = 1$（单位梯度）
- 零等值面：$\phi(x) = 0$ 表示物体表面
- 内部：$\phi(x) < 0$
- 外部：$\phi(x) > 0$

### 2.2 三次 SDF 插值

使用**三线性插值**（Trilinear Interpolation）构建平滑 SDF：

```
φ(x,y,z) = Σᵢ Σⱼ Σₖ cᵢⱼₖ · Bᵢ(x) · Bⱼ(y) · Bₖ(z)
```

其中 $B_i$ 是三次 B-spline 基函数：

```
B(t) = (1/6) × {
    (2-t)³ - 4(1-t)³          0 ≤ t < 1
    (2-t)³                     1 ≤ t < 2
    0                          otherwise
}
```

### 2.3 可微接触力模型

**平滑屏障函数**：
```
b(d) = { -κ·(d-d̂)²·ln(d/d̂)   如果 d < d̂
        0                    如果 d ≥ d̂
```

其中：
- $d = \phi(x)$：SDF 距离值
- $d̂$：屏障距离阈值
- $κ$：刚度系数

**接触力**：
```
F_contact = -∇b(d) = -b'(d) · ∇φ(x)
```

**接触力梯度（Jacobian）**：
```
∂F/∂x = -b''(d) · ∇φ ⊗ ∇φ - b'(d) · ∇²φ
```

## 4. 实现细节

### 4.1 三次 SDF 插值实现

```cpp
// src/DifferentiablePhysics/CubicSDFGrid.cpp

template<typename T>
T CubicSDFGrid::evaluate(const Eigen::Matrix<T, 3, 1>& x) const {
    // 转换到网格坐标
    Eigen::Matrix<T, 3, 1> grid_coord = (x - origin_.cast<T>()) / spacing_.cast<T>();
    
    // 获取整数索引和小数部分
    Eigen::Vector3i idx = grid_coord.template cast<int>();
    Eigen::Matrix<T, 3, 1> frac = grid_coord - idx.cast<T>();
    
    // 三线性插值
    T result = T(0);
    for (int i = 0; i < 4; ++i) {
        for (int j = 0; j < 4; ++j) {
            for (int k = 0; k < 4; ++k) {
                // 获取网格节点值
                Eigen::Vector3i node_idx = idx + Eigen::Vector3i(i-1, j-1, k-1);
                
                // 边界检查
                if (node_idx.minCoeff() < 0 || 
                    node_idx[0] >= dimensions_[0] ||
                    node_idx[1] >= dimensions_[1] ||
                    node_idx[2] >= dimensions_[2]) {
                    continue;
                }
                
                int flat_idx = node_idx[0] + dimensions_[0] * (node_idx[1] + dimensions_[1] * node_idx[2]);
                T node_value = T(sdf_values_[flat_idx]);
                
                // B-spline 权重
                T weight = T(bspline(frac[0] + (1-i))) * 
                          T(bspline(frac[1] + (1-j))) * 
                          T(bspline(frac[2] + (1-k)));
                
                result += weight * node_value;
            }
        }
    }
    
    return result;
}

// B-spline 基函数
double CubicSDFGrid::bspline(double t) {
    t = std::abs(t);
    if (t >= 2.0) return 0.0;
    if (t >= 1.0) {
        double u = 2.0 - t;
        return (1.0/6.0) * u * u * u;
    }
    return (1.0/6.0) * ((2.0-t)*(2.0-t)*(2.0-t) - 4.0*(1.0-t)*(1.0-t)*(1.0-t));
}
```

### 4.2 平滑屏障函数实现

```cpp
// src/DifferentiablePhysics/SmoothContactModel.cpp

template<typename T>
T SmoothContactModel::barrierFunction(T d) const {
    T dhat = T(params_.dhat);
    T kappa = T(params_.stiffness);
    
    if (d >= dhat) {
        return T(0);
    }
    
    // IPC 平滑屏障：-κ·(d-d̂)²·ln(d/d̂)
    T diff = d - dhat;
    T ratio = d / dhat;
    
    // 数值稳定性处理
    using std::log;
    T log_term = log(ratio + T(1e-10));
    
    return -kappa * diff * diff * log_term;
}

template<typename T>
T SmoothContactModel::barrierDerivative(T d) const {
    T dhat = T(params_.dhat);
    T kappa = T(params_.stiffness);
    
    if (d >= dhat) {
        return T(0);
    }
    
    // b'(d) = -κ·[2(d-d̂)·ln(d/d̂) + (d-d̂)²/d]
    T diff = d - dhat;
    using std::log;
    T log_term = log(d / dhat + T(1e-10));
    
    return -kappa * (T(2) * diff * log_term + diff * diff / d);
}

template<typename T>
Eigen::Matrix<T, 3, 1> SmoothContactModel::computeContactForce(
    int object_id,
    const Eigen::Matrix<T, 3, 1>& x) const {
    
    auto it = objects_.find(object_id);
    if (it == objects_.end()) {
        return Eigen::Matrix<T, 3, 1>::Zero();
    }
    
    // 计算 SDF 值和梯度
    T d = it->second->evaluate(x);
    
    // 只在接触时计算力
    if (d >= T(params_.dhat)) {
        return Eigen::Matrix<T, 3, 1>::Zero();
    }
    
    // 计算 SDF 梯度（法线方向）
    Eigen::Matrix<T, 3, 1> normal = it->second->gradient(x).cast<T>();
    normal.normalize();
    
    // 接触力 = -b'(d) · n
    T force_magnitude = -barrierDerivative(d);
    return force_magnitude * normal;
}
```

// src/DifferentiablePhysics/DifferentiableDynamics.cpp

template<typename T>
Eigen::Matrix<T, Eigen::Dynamic, 1> DifferentiableDynamics::step(
    const Eigen::Matrix<T, Eigen::Dynamic, 1>& state,
    const Eigen::Matrix<T, Eigen::Dynamic, 1>& control,
    T dt) {
    
    // 状态分解：位置 + 速度
    int n = state.size() / 2;
    Eigen::Matrix<T, Eigen::Dynamic, 1> q = state.head(n);  // 位置
    Eigen::Matrix<T, Eigen::Dynamic, 1> v = state.tail(n);  // 速度
    
    // 隐式欧拉积分
    // v_{t+1} = v_t + dt * a(q_{t+1}, v_{t+1})
    // q_{t+1} = q_t + dt * v_{t+1}
    
    // 使用牛顿法求解非线性方程
    Eigen::Matrix<T, Eigen::Dynamic, 1> q_new = q;
    Eigen::Matrix<T, Eigen::Dynamic, 1> v_new = v;
    
    for (int iter = 0; iter < 10; ++iter) {
        // 计算加速度（包含接触力）
        Eigen::Matrix<T, Eigen::Dynamic, 1> accel = 
            computeAcceleration(q_new, v_new, control);
        
        // 隐式欧拉残差
        Eigen::Matrix<T, Eigen::Dynamic, 1> residual_v = v_new - v - dt * accel;
        Eigen::Matrix<T, Eigen::Dynamic, 1> residual_q = q_new - q - dt * v_new;
        
        // 检查收敛
        if (residual_v.norm() < T(1e-6) && residual_q.norm() < T(1e-6)) {
            break;
        }

    }

template<typename T>
Eigen::Matrix<T, Eigen::Dynamic, 1> DifferentiableDynamics::computeAcceleration(
    const Eigen::Matrix<T, Eigen::Dynamic, 1>& q,
    const Eigen::Matrix<T, Eigen::Dynamic, 1>& v,
    const Eigen::Matrix<T, Eigen::Dynamic, 1>& control) {
    
    int n = q.size();
    Eigen::Matrix<T, Eigen::Dynamic, 1> accel = Eigen::Matrix<T, Eigen::Dynamic, 1>::Zero(n);
    
    // 重力
    for (int i = 0; i < n / 3; ++i) {
        accel[3*i + 1] += T(-9.81);  // y 方向重力
    }
    
    // 接触力
    for (int i = 0; i < n / 3; ++i) {
        Eigen::Matrix<T, 3, 1> pos = q.segment<3>(3*i);
        
        // 对每个物体计算接触力
        for (const auto& [obj_id, sdf] : contact_model_->getObjects()) {
            Eigen::Matrix<T, 3, 1> contact_force = 
                contact_model_->computeContactForce(obj_id, pos);
            
            // F = ma, 所以 a = F/m
            T mass = T(1.0);  // 假设单位质量
            accel.segment<3>(3*i) += contact_force / mass;
        }
    }
    
    // 控制力
    accel += control;
    
    return accel;
}

## 5. 与神经网络集成

### 5.1 PyTorch 绑定

```cpp
// python/bindings/differentiable_physics.cpp

#include <torch/extension.h>
#include "NexDynIPC/DifferentiablePhysics/DifferentiableDynamics.h"

namespace py = pybind11;
using namespace NexDynIPC::DifferentiablePhysics;

// 自定义 PyTorch 自动微分函数
class DifferentiablePhysicsFunction : public torch::autograd::Function<DifferentiablePhysicsFunction> {
public:
    static torch::Tensor forward(
        torch::autograd::AutogradContext* ctx,
        torch::Tensor initial_state,
        torch::Tensor controls,
        double dt,
        int num_steps,
        std::shared_ptr<DifferentiableDynamics> solver) {
        
        // 转换到 Eigen
        Eigen::VectorXd x0 = torchToEigen(initial_state);
        std::vector<Eigen::VectorXd> u_seq = torchToEigenVector(controls);
        
        // 前向模拟
        autodiff::VectorXvar<> x0_ad = x0.cast<autodiff::var>();
        std::vector<autodiff::VectorXvar<>> u_ad;
        for (const auto& u : u_seq) {
            u_ad.push_back(u.cast<autodiff::var>());
        }
        
        auto result = solver->forward(x0_ad, u_ad, autodiff::var(dt), num_steps);
        
        // 保存用于反向传播
        ctx->saved_data["solver"] = reinterpret_cast<intptr_t>(solver.get());
        ctx->saved_data["dt"] = dt;
        ctx->saved_data["num_steps"] = num_steps;
        
        // 转换回 PyTorch
        return eigenToTorch(result.states.back());
    }
    
    static torch::autograd::tensor_list backward(
        torch::autograd::AutogradContext* ctx,
        torch::autograd::tensor_list grad_outputs) {
        
        // 获取保存的数据
        auto solver = reinterpret_cast<DifferentiableDynamics*>(
            ctx->saved_data["solver"].toInt());
        
        // 反向传播梯度
        Eigen::VectorXd grad_loss = torchToEigen(grad_outputs[0]);
        auto grad_result = solver->backward(/* ... */, grad_loss);
        
        return {
            eigenToTorch(grad_result.dL_dinitial_state),
            eigenToTorch(grad_result.dL_dparameters),
            torch::Tensor(),
            torch::Tensor(),
            torch::Tensor()
        };
    }
};

// Python 绑定
PYBIND11_MODULE(differentiable_physics, m) {
    m.def("simulate", [](torch::Tensor initial_state,
                         torch::Tensor controls,
                         double dt,
                         int num_steps,
                         std::shared_ptr<DifferentiableDynamics> solver) {
        return DifferentiablePhysicsFunction::apply(
            initial_state, controls, dt, num_steps, solver);
    });
}
```
