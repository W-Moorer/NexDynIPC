# Rigid-IPC 开环控制详细分析

## 1. 概述

Rigid-IPC 是一个基于增量势能接触（Incremental Potential Contact, IPC）框架的刚体动力学模拟系统。该系统的"开环控制"主要体现在**运动学刚体（Kinematic Bodies）**的控制机制上，通过预设的位姿序列或速度指令来驱动刚体运动，而不受物理约束（如碰撞、摩擦）的反馈影响。

---

## 3. 开环控制的核心机制

### 3.1 刚体类型与开环控制的关系

Rigid-IPC 定义了三种刚体类型（`RigidBodyType` 枚举）：

```cpp
enum RigidBodyType { STATIC, KINEMATIC, DYNAMIC };
```

| 类型 | 说明 | 控制方式 |
|------|------|----------|
| `STATIC` | 静态刚体 | 位置和速度始终为零，完全固定 |
| `KINEMATIC` | 运动学刚体 | **开环控制的核心**，通过预设位姿或速度驱动 |
| `DYNAMIC` | 动态刚体 | 受物理定律支配，通过力和碰撞响应运动 |

### 3.2 运动学刚体的开环控制实现

#### 3.2.1 数据结构定义

在 `RigidBody` 类中（`physics/rigid_body.hpp`），运动学刚体的控制数据定义如下：

```cpp
class RigidBody {
    // ... 其他成员 ...
    
    // ----------------------------------------------------------------
    // Scripted kinematic motion (脚本化运动学运动)
    // ----------------------------------------------------------------
    double kinematic_max_time;           // 运动学控制的最大持续时间
    std::deque<PoseD> kinematic_poses;   // 预设位姿序列（用于关键帧控制）
};
```

#### 3.2.2 两种开环控制模式

Rigid-IPC 支持两种运动学刚体的开环控制模式：

**模式一：速度驱动（Velocity-Controlled）**

通过设置刚体的 `velocity` 属性来实现连续运动：

```cpp
// 在 DistanceBarrierRBProblem::init_augmented_lagrangian() 中
for (int i = 0; i < num_bodies(); i++) {
    if (m_assembler[i].kinematic_poses.size()) {
        // 使用预设位姿序列
        const PoseD& pose = m_assembler[i].kinematic_poses.front();
        x_pred.segment(ndof * i, pos_ndof) = pose.position;
        x_pred.segment(ndof * i + pos_ndof, rot_ndof) = pose.rotation;
    } else {
        // 使用速度预测
        x_pred.segment(ndof * i, pos_ndof) +=
            timestep() * m_assembler[i].velocity.position;
        x_pred.segment(ndof * i + pos_ndof, rot_ndof) +=
            timestep() * m_assembler[i].velocity.rotation;
    }
}
```

**模式二：关键帧驱动（Keyframe-Controlled）**

通过 `kinematic_poses` 队列实现离散位姿关键帧：

```cpp
// 在 DistanceBarrierRBProblem::step_kinematic_bodies() 中
void DistanceBarrierRBProblem::step_kinematic_bodies()
{
    for (int i = 0; i < num_bodies(); i++) {
        if (m_assembler[i].type == RigidBodyType::KINEMATIC) {
            if (m_assembler[i].kinematic_max_time < 0) {
                m_assembler[i].convert_to_static();  // 超时后转为静态
            } else {
                m_assembler[i].kinematic_max_time -= timestep();
                if (m_assembler[i].kinematic_poses.size()) {
                    m_assembler[i].kinematic_poses.pop_front();  // 消费关键帧
                }
            }
        }
    }
}
```

### 3.3 增广拉格朗日约束实现

运动学刚体的开环控制通过**增广拉格朗日方法（Augmented Lagrangian）**强制实现，确保优化求解器将运动学刚体的位姿约束在预设轨迹上。

#### 3.3.1 初始化

```cpp
void DistanceBarrierRBProblem::init_augmented_lagrangian()
{
    int ndof = PoseD::dim_to_ndof(dim());
    int pos_ndof = PoseD::dim_to_pos_ndof(dim());
    int rot_ndof = PoseD::dim_to_rot_ndof(dim());

    // 初始化惩罚系数
    linear_augmented_lagrangian_penalty = 1e3;
    angular_augmented_lagrangian_penalty = 1e3;
    
    // 初始化拉格朗日乘子
    linear_augmented_lagrangian_multiplier.setZero(pos_ndof * num_kinematic_bodies);
    angular_augmented_lagrangian_multiplier.setZero(rot_ndof * num_kinematic_bodies, rot_ndof);

    // 标记已满足的约束
    is_dof_satisfied.setZero(x0.size());
    for (int i = 0; i < num_bodies(); i++) {
        if (m_assembler[i].type == RigidBodyType::STATIC) {
            is_dof_satisfied.segment(ndof * i, ndof).setOnes();
        }
    }
}
```

#### 3.3.2 增广拉格朗日势能计算

```cpp
double DistanceBarrierRBProblem::compute_augmented_lagrangian(
    const Eigen::VectorXd& x,
    Eigen::VectorXd& grad,
    Eigen::SparseMatrix<double>& hess,
    bool compute_grad,
    bool compute_hess)
{
    // 线性部分势能
    // potential = κ_q/2 * m * ||q - q_pred||² - √m * λ·(q - q_pred)
    for (size_t i = 0, ki = 0; i < num_bodies(); i++) {
        if (m_assembler[i].type != RigidBodyType::KINEMATIC) continue;

        double m = m_assembler[i].mass;
        const auto& lambda = linear_augmented_lagrangian_multiplier.segment(ki * pos_ndof, pos_ndof);
        const auto& q = x.segment(i * ndof, pos_ndof);
        const auto& q_pred = x_pred.segment(i * ndof, pos_ndof);

        potential += kappa_q / 2 * m * (q - q_pred).squaredNorm()
                   - sqrt(m) * lambda.dot(q - q_pred);
        
        if (compute_grad) {
            grad.segment(i * ndof, pos_ndof) = kappa_q * m * (q - q_pred) - sqrt(m) * lambda;
        }
        // ... Hessian 计算 ...
    }

    // 角向部分势能（3D使用旋转矩阵，2D使用标量角度）
    // potential = κ_Q/2 * ||(Q - Q_pred) * J||² - Λ·(Q - Q_pred) * J_sqrt
    // ...
}
```

#### 3.3.3 乘子更新策略

```cpp
void DistanceBarrierRBProblem::update_augmented_lagrangian(const Eigen::VectorXd& x)
{
    // 计算收敛进度 η
    double eta_q = compute_linear_augment_lagrangian_progress(x);
    double eta_Q = compute_angular_augment_lagrangian_progress(x);

    // 线性部分更新
    if (eta_q >= 0.999) {
        // 收敛：固定运动学自由度
        for (size_t i = 0; i < num_bodies(); i++) {
            if (m_assembler[i].type == RigidBodyType::KINEMATIC) {
                is_dof_satisfied.segment(ndof * i, pos_ndof).setOnes();
            }
        }
    } else if (eta_q < 0.99 && linear_augmented_lagrangian_penalty < 1e8) {
        // 未收敛：增加惩罚系数 κ_q
        linear_augmented_lagrangian_penalty *= 2;
    } else {
        // 增加拉格朗日乘子 λ
        for (size_t i = 0, ki = 0; i < num_bodies(); i++) {
            if (m_assembler[i].type == RigidBodyType::KINEMATIC) {
                linear_augmented_lagrangian_multiplier.segment(ki * pos_ndof, pos_ndof) -=
                    linear_augmented_lagrangian_penalty * sqrt(m_assembler[i].mass)
                    * (x.segment(ndof * i, pos_ndof) - x_pred.segment(ndof * i, pos_ndof));
                ki++;
            }
        }
    }
    
    // 角向部分类似更新 ...
}
```

### 4.2 关键耦合方程

#### 4.2.1 运动学约束耦合

运动学刚体的位姿作为等式约束嵌入优化问题：

```
minimize    f(x) = E(x) + κ·B(x) + D(x)
subject to  q_i = q_pred_i,  ∀i ∈ KinematicBodies
            Q_i = Q_pred_i,  ∀i ∈ KinematicBodies
```

通过增广拉格朗日方法转化为无约束问题：

```
L_AL(x, λ, κ) = f(x) 
    + Σᵢ [κ_q/2·mᵢ·||qᵢ - q_predᵢ||² - √mᵢ·λᵢ·(qᵢ - q_predᵢ)]  (线性部分)
    + Σᵢ [κ_Q/2·||(Qᵢ - Q_predᵢ)·Jᵢ||² - Λᵢ·(Qᵢ - Q_predᵢ)·J_sqrtᵢ]  (角向部分)
```

#### 4.2.2 碰撞-控制耦合

运动学刚体与动态刚体的碰撞通过距离障碍函数处理：

```
B(x) = Σ_{k ∈ C} b(d(x_k))

其中:
- b(d) = -(d - d̂)²·ln(d/d̂)  (对数障碍函数)
- d(x_k) = ||Vᵢ(x) - Vⱼ(x)||  (顶点间距离)
- d̂ = barrier_activation_distance  (障碍激活距离)
```

碰撞力计算（梯度）：

```
F_barrier = -κ · ∇B(x) = -κ · Σ_{k ∈ C} b'(d(x_k)) · ∇d(x_k)
```

#### 4.2.3 摩擦-碰撞耦合

摩擦势能与碰撞约束集耦合：

```
D(x) = Σ_{k ∈ C_f} μ·κ·b(d(x_k))·(||u_T||² / (||u_T||² + ε_v²·h²))

其中:
- u_T = u - (u·n)·n  (切向相对位移)
- ε_v = static_friction_speed_bound  (静摩擦速度阈值)
- h = timestep  (时间步长)
```

### 4.3 耦合数据流

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          时间步进流程                                     │
└─────────────────────────────────────────────────────────────────────────┘

Time Step t → t+1:

1. 保存当前状态
```

## 5. 理论基础

### 5.1 增量势能接触 (IPC) 理论

IPC 框架将接触问题转化为优化问题，核心思想是通过**对数障碍函数**在碰撞发生前产生排斥力，确保无穿透。

#### 5.1.1 障碍函数

```
b(d) = {
    -(d - d̂)²·ln(d/d̂),  0 < d < d̂
    0,                  d ≥ d̂
}
```

特性：
- 当 d → 0⁺ 时，b(d) → +∞（防止穿透）
- 当 d = d̂ 时，b(d) = 0（障碍边界）
- 当 d > d̂ 时，b(d) = 0（无障碍作用）

#### 5.1.2 自适应障碍刚度

```cpp
// IPCSolver::init_solve 中初始化 κ
kappa = initial_barrier_stiffness(
    bbox_diagonal, dhat, average_mass, grad_E, grad_B, max_barrier_stiffness);

// IPCSolver::post_step_update 中自适应更新
update_barrier_stiffness(
    prev_min_distance, min_distance, max_barrier_stiffness, kappa, bbox_diagonal, dhat_epsilon);
```

### 5.2 刚体动力学理论

#### 5.2.1 位姿表示

- **2D**: 位姿 `Pose = [x, y, θ]` (3自由度)
- **3D**: 位姿 `Pose = [x, y, z, θ_x, θ_y, θ_z]` (6自由度)

旋转使用**旋转向量**表示，通过 `construct_rotation_matrix()` 转换为旋转矩阵。

#### 5.2.2 能量积分方法

Rigid-IPC 支持三种能量积分方法（`BodyEnergyIntegrationMethod` 枚举）：

| 方法 | 公式 | 特性 |
|------|------|------|
| Implicit Euler | `v₁ = (q₁ - q₀) / h` | 一阶精度，稳定 |
| Implicit Newmark | `v₁ = 2(q₁ - q₀)/h - v₀` | 二阶精度，能量守恒 |
| Stabilized Newmark | 带稳定项的 Newmark | 抑制高频振荡 |

#### 5.2.3 旋转更新（3D）

```cpp
// 计算旋转矩阵差
Eigen::Matrix3d R = Q₁ * Q₀.transpose();

// 提取角速度
Eigen::AngleAxisd omega(R);
rb.velocity.rotation = omega.angle() / h * R₀.transpose() * omega.axis();
```

### 5.3 优化理论

#### 5.3.1 目标函数组成

```
f(x) = E(x)/m_avg + κ·B(x)/m_avg + D(x)/m_avg + AL(x)/m_avg

其中:
- E(x): 刚体能量项（惯性 + 外力 + 重力）
- B(x): 距离障碍项（防止穿透）
- D(x): 摩擦耗散项（切向阻尼）
- AL(x): 增广拉格朗日项（运动学约束）
- m_avg: 平均质量（归一化）
```

#### 5.3.2 牛顿迭代

```
x_{k+1} = x_k + α·Δx

其中 Δx 通过求解线性系统得到:
H·Δx = -g

H = ∇²f(x)  (Hessian矩阵)
g = ∇f(x)   (梯度向量)
α = line_search(...)  (步长)
```

## 6. 代码关键路径分析

### 6.1 模拟步进主流程

```cpp
// DistanceBarrierRBProblem::simulation_step
void simulation_step(bool& had_collisions, bool& _has_intersections, bool solve_collisions)
{
    // 1. 保存前一时刻状态
    for (size_t i = 0; i < num_bodies(); i++) {
        m_assembler[i].pose_prev = m_assembler[i].pose;
        m_assembler[i].velocity_prev = m_assembler[i].velocity;
    }

    // 2. 更新自由度
    update_dof();

    // 3. 重置碰撞标志
    m_had_collisions = false;
    m_num_contacts = 0;

    // 4. 控制是否求解碰撞（开环控制时可禁用）
    this->m_use_barriers = solve_collisions;

    // 5. 求解约束
    update_constraints();
    opt_result = solve_constraints();
    
    // 6. 执行步进
    _has_intersections = take_step(opt_result.x);
    
    // 7. 更新运动学刚体
    step_kinematic_bodies();
    
    had_collisions = m_had_collisions;
}
```

### 6.2 能量项计算

```cpp
// DistanceBarrierRBProblem::compute_body_energy
template <typename T>
T compute_body_energy(const RigidBody& body, const Pose<T>& pose, const VectorMax6d& grad_barrier_t0)
{
    T energy(0.0);
    double h = timestep();

    // 线性能量: ½mqᵀq - mqᵀ(qᵗ + h(q̇ᵗ + h(g + f/m + ½∇B(qᵗ)/m)))
    if (!body.is_dof_fixed.head(pose.pos_ndof()).all()) {
        VectorMax3<T> q = pose.position;
        const VectorMax3d& q_t0 = body.pose.position;
        const VectorMax3d& qdot_t0 = body.velocity.position;
        VectorMax3d qddot_t0 = gravity + body.force.position / body.mass;
        
        energy += 0.5 * body.mass * q.dot(q)
                - body.mass * q.dot(q_t0 + h * (qdot_t0 + h * qddot_t0));
    }

    // 旋转能量: ½tr(QJQᵀ) - tr(Q(J(Qᵗ + hQ̇ᵗ + h²Aᵗ)ᵀ + h²[τ]))
    if (!body.is_dof_fixed.tail(pose.rot_ndof()).all()) {
        if (dim() == 3) {
            Matrix3<T> Q = pose.construct_rotation_matrix();
            Eigen::Matrix3d Q_t0 = body.pose.construct_rotation_matrix();
            Eigen::Matrix3d Qdot_t0 = body.Qdot;
            DiagonalMatrix3d J = compute_J(body.moment_of_inertia);
            
            energy += 0.5 * (Q * J * Q.transpose()).trace();
            energy -= (Q * J * (Q_t0 + h * Qdot_t0).transpose()).trace();
        } else {
            // 2D旋转能量
            T theta = pose.rotation[0];
            double theta_t0 = body.pose.rotation[0];
            double theta_dot_t0 = body.velocity.rotation[0];
            double I = body.moment_of_inertia[0];
            
            energy += 0.5 * I * theta * theta 
                    - I * theta * (theta_t0 + h * theta_dot_t0);
        }
    }
    
    return energy;
}
```

### 6.3 碰撞约束构造

```cpp
// DistanceBarrierConstraint::construct_constraint_set
void construct_constraint_set(
    const RigidBodyAssembler& bodies,
    const PosesD& poses,
    Constraints& constraint_set) const
{
    // 1. 宽阶段检测 (Broad Phase)
    Candidates candidates;
    detect_collision_candidates(bodies, poses, candidates);
    
    // 2. 窄阶段检测 (Narrow Phase)
    for (const auto& candidate : candidates) {
        double distance = compute_distance(candidate, bodies, poses);
        if (distance < barrier_activation_distance()) {
            // 添加活跃约束
            constraint_set.add_constraint(candidate, distance);
        }
    }
}
```

## 8. 总结

### 8.1 开环控制特点

1. **预设轨迹**: 运动学刚体的运动完全由用户预设的位姿序列或速度决定
2. **无反馈**: 不受碰撞、摩擦等物理力的反馈影响（但会对动态刚体产生作用力）
3. **硬约束**: 通过增广拉格朗日方法将运动学约束作为硬约束强制执行
4. **时间控制**: 支持时间限制，超时后自动转为静态刚体

### 8.2 耦合关系总结

| 耦合类型 | 耦合机制 | 数学表达 |
|----------|----------|----------|
| 控制-优化 | 增广拉格朗日 | `AL(x) = κ/2·||x - x_pred||² - λ·(x - x_pred)` |
| 优化-碰撞 | 距离障碍 | `B(x) = Σ b(d(x_k))` |
| 碰撞-摩擦 | 摩擦势能 | `D(x) = Σ μ·b(d)·f(u_T)` |
| 优化-动力学 | 能量最小化 | `E(x) = ½mv² - mgh` |

### 8.3 应用场景

- **机械臂控制**: 预设关节轨迹，模拟与环境的交互
- **传送带模拟**: 恒定速度驱动，输送动态物体
- **关键帧动画**: 离散位姿序列驱动，实现精确控制
- **边界条件**: 静态/运动学边界作为动态系统的约束
