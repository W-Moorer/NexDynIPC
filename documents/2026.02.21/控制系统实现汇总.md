# NexDynIPC 控制系统实现汇总

## 文档说明

本文档详细汇总了 NexDynIPC 项目中已实现的控制系统，包括：
1. **位移控制（开环控制）** - 基于轨迹的运动学控制
2. **力控制** - 基于外力的动力学控制

所有实现均包含完整代码、使用示例和测试验证。

## 一、位移控制（开环控制）

### 1.1 系统架构

位移控制通过预设轨迹直接控制刚体的位姿，不涉及力的计算。

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   Trajectory    │────▶│ KinematicBody   │────▶│  RigidBody      │
│   (轨迹定义)     │     │ (运动学刚体)     │     │  (物理刚体)      │
└─────────────────┘     └─────────────────┘     └─────────────────┘
         │                                               ▲
         │                                               │
         └───────────────────────────────────────────────┘
                    OpenLoopController
                    (开环控制器)
```

### 1.2 核心组件

#### 1.2.1 Trajectory（轨迹基类）

**文件位置**：`include/NexDynIPC/Control/Trajectory.h`

**功能**：定义随时间变化的目标位姿接口。

**类层次**：
```cpp
class Trajectory {
public:
    virtual void getPose(double t, Vector3d& pos, Quaterniond& ori) const = 0;
    virtual void getVelocity(double t, Vector3d& vel, Vector3d& ang_vel) const = 0;
};
```

**实现类**：

| 类名 | 功能 | 公式 |
|------|------|------|
| `FixedTrajectory` | 固定位姿 | `p(t) = p_0, q(t) = q_0` |
| `LinearTrajectory` | 匀速直线 | `p(t) = p_0 + v*t` |
| `KeyframeTrajectory` | 关键帧插值 | 线性插值 + SLERP |
| `FunctionTrajectory` | 函数式轨迹 | 自定义回调函数 |

**使用示例**：
```cpp
// 固定轨迹
auto fixed_traj = std::make_shared<FixedTrajectory>(
    Vector3d(1.0, 2.0, 3.0),                    // 位置
    Quaterniond::Identity()                      // 朝向
);

// 线性轨迹
auto linear_traj = std::make_shared<LinearTrajectory>(
    Vector3d(0.0, 0.0, 0.0),                    // 起始位置
    Vector3d(1.0, 0.0, 0.0)                     // 速度 (1m/s X方向)
);

// 关键帧轨迹
auto keyframe_traj = std::make_shared<KeyframeTrajectory>();
keyframe_traj->addKeyframe({0.0, Vector3d(0,0,0), Quaterniond::Identity()});
keyframe_traj->addKeyframe({1.0, Vector3d(1,0,0), Quaterniond::Identity()});
keyframe_traj->addKeyframe({2.0, Vector3d(1,1,0), Quaterniond::Identity()});

// 函数式轨迹（圆周运动）
auto func_traj = std::make_shared<FunctionTrajectory>(
    [](double t, Vector3d& pos, Quaterniond& ori) {
        pos = Vector3d(cos(t), sin(t), 0.0);
        ori = Quaterniond::Identity();
    }
);
```

#### 1.2.2 KinematicBody（运动学刚体）

**文件位置**：`include/NexDynIPC/Control/KinematicBody.h`

**功能**：继承自 RigidBody，通过轨迹控制位姿而非物理求解。

**核心方法**：
```cpp
class KinematicBody : public RigidBody {
public:
    void setTrajectory(std::shared_ptr<Trajectory> trajectory);
    void updateTargetPose(double t);      // 从轨迹更新目标位姿
    void applyTargetPose();                // 应用目标位姿到实际状态
    
    Vector3d getTargetPosition() const;
    Quaterniond getTargetOrientation() const;
    Vector3d getTargetVelocity() const;
};
```

**使用示例**：
```cpp
auto body = std::make_shared<KinematicBody>();
body->setTrajectory(linear_traj);
body->updateTargetPose(2.0);  // t=2s
body->applyTargetPose();       // 应用到位姿
// 现在 body->position = (2, 0, 0)
```

#### 1.2.3 OpenLoopController（开环控制器）

**文件位置**：`include/NexDynIPC/Control/OpenLoopController.h`

**功能**：管理多个运动学刚体的轨迹执行。

**核心方法**：
```cpp
class OpenLoopController {
public:
    void addKinematicBody(std::shared_ptr<KinematicBody> body);
    void removeKinematicBody(std::shared_ptr<KinematicBody> body);
    
    void preSolveUpdate(double t, double dt);   // 预求解：更新目标位姿
    void postSolveUpdate();                      // 后求解：应用位姿
    void applyTrajectories(double t);            // 直接应用轨迹
    
    size_t getBodyCount() const;
    void clear();
};
```

**使用示例**：
```cpp
// 创建控制器
auto controller = std::make_shared<OpenLoopController>();

// 创建运动学刚体
auto platform = std::make_shared<KinematicBody>();
platform->setTrajectory(circular_traj);

// 添加到控制器
controller->addKinematicBody(platform);

// 仿真循环
double t = 0.0, dt = 0.01;
for (int i = 0; i < 1000; ++i) {
    controller->preSolveUpdate(t, dt);
    solver->step(world, dt);
    controller->postSolveUpdate();
    t += dt;
}
```

## 二、力控制

### 2.1 系统架构

力控制通过在刚体上施加外力/力矩来控制运动，遵循牛顿-欧拉方程。

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  Force Command  │────▶│ ExternalForceForm│────▶│  Simulation     │
│  (力指令)        │     │ (外力形式)       │     │  (物理求解)      │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                                │
                                ▼
                        ┌─────────────────┐
                        │  Gradient: -F   │
                        │  (负梯度=力)     │
                        └─────────────────┘
```

### 2.2 核心组件

#### 2.2.1 ExternalForceForm（外力形式）

**文件位置**：`include/NexDynIPC/Control/ExternalForceForm.h`

**功能**：在刚体上施加外部力/力矩，通过 Form 系统集成到 IPC 求解器。

**物理原理**：
```
势能 E = -F·x
梯度 ∇E = -F

在牛顿求解器中：
grad += ∇E = -F  (负梯度方向为力的方向)
```

**核心方法**：
```cpp
class ExternalForceForm : public Form {
public:
    void addBody(std::shared_ptr<RigidBody> body);
    void removeBody(std::shared_ptr<RigidBody> body);
    
    // 施加力（世界坐标系）
    void setForce(std::shared_ptr<RigidBody> body, 
                  const Vector3d& force,
                  const Vector3d& local_point = Vector3d::Zero());
    
    // 施加力矩
    void setTorque(std::shared_ptr<RigidBody> body, 
                   const Vector3d& torque);
    
    // 同时施加力和力矩
    void setForceAndTorque(std::shared_ptr<RigidBody> body,
                           const Vector3d& force,
                           const Vector3d& torque,
                           const Vector3d& local_point = Vector3d::Zero());
    
    // 清除力
    void clearForces(std::shared_ptr<RigidBody> body);
    void clearAllForces();
    
    // Form 接口
    double value(const VectorXd& x) const override;
    void gradient(const VectorXd& x, VectorXd& grad) const override;
    void hessian(const VectorXd& x, std::vector<Triplet>& triplets) const override;
};
```

**使用示例**：
```cpp
// 创建力控制形式
auto force_form = std::make_shared<ExternalForceForm>();

// 添加刚体
auto box = std::make_shared<RigidBody>();
box->mass = 5.0;
force_form->addBody(box);

// 施加恒定力
force_form->setForce(box, Vector3d(10.0, 0.0, 0.0));  // X方向10N

// 施加力矩
force_form->setTorque(box, Vector3d(0.0, 0.0, 2.0));  // Z方向2Nm

// 同时施加力和力矩
force_form->setForceAndTorque(
    box,
    Vector3d(10.0, 0.0, 0.0),   // 力
    Vector3d(0.0, 2.0, 0.0),    // 力矩
    Vector3d::Zero()            // 作用点（质心）
);

// 添加到World
world->addForm(force_form);

// 在仿真中，力会自动应用
solver->step(world, dt);
```

**时变力示例**：
```cpp
// 仿真循环中更新力
double t = 0.0, dt = 0.01;
for (int i = 0; i < 1000; ++i) {
    // 正弦变化力
    double force_mag = 10.0 * sin(t);
    force_form->setForce(box, Vector3d(force_mag, 0.0, 0.0));
    
    solver->step(world, dt);
    t += dt;
}
```

#### 2.2.2 JointActuator（关节驱动器）- 接口定义

**文件位置**：`include/NexDynIPC/Control/JointActuator.h`

**功能**：在关节上施加驱动力矩，支持多种控制模式。

**控制模式**：
```cpp
enum class ControlMode {
    TORQUE_CONTROL,    // 直接力矩控制
    POSITION_CONTROL,  // PD位置控制
    VELOCITY_CONTROL   // PD速度控制
};
```

**核心方法**：
```cpp
class JointActuator {
public:
    // 设置控制模式
    void setControlMode(ControlMode mode);
    
    // 力矩控制
    void setTargetTorque(double torque);
    
    // 位置控制（PD）
    void setTargetPosition(double position);
    void setPDGains(double kp, double kd);
    
    // 速度控制（PD）
    void setTargetVelocity(double velocity);
    
    // 力矩限制
    void setTorqueLimits(double min_torque, double max_torque);
    
    // 计算并应用控制
    double computeControlOutput();
};
```

**PD控制算法**：
```cpp
double JointActuator::computeControlOutput() {
    switch (mode_) {
        case ControlMode::TORQUE_CONTROL:
            return saturateTorque(target_torque_);
            
        case ControlMode::POSITION_CONTROL: {
            double pos_error = target_position_ - getCurrentPosition();
            double vel_error = -getCurrentVelocity();
            double tau = kp_ * pos_error + kd_ * vel_error;
            return saturateTorque(tau);
        }
        
        case ControlMode::VELOCITY_CONTROL: {
            double vel_error = target_velocity_ - getCurrentVelocity();
            double tau = kp_ * vel_error;
            return saturateTorque(tau);
        }
    }
}
```

## 三、控制方式对比

| 特性 | 位移控制（开环） | 力控制 |
|------|----------------|-------|
| **控制量** | 位置 p(t), 朝向 q(t) | 力 F, 力矩 τ |
| **物理响应** | 无（运动学） | 有（动力学） |
| **实现方式** | KinematicBody + Trajectory | ExternalForceForm |
| **适用场景** | 预设路径、动画 | 接触力、夹取力 |
| **碰撞响应** | 硬约束（穿透） | 软约束（物理正确） |
| **计算成本** | 低 | 中（需物理求解） |

## 四、综合使用示例

### 4.1 机械臂控制

```cpp
// 创建World
auto world = std::make_shared<World>();

// ========== 位移控制：基座 ==========
auto base = std::make_shared<KinematicBody>();
base->setTrajectory(std::make_shared<FixedTrajectory>(
    Vector3d(0, 0, 0), Quaterniond::Identity()
));
world->addBody(base);

// ========== 力控制：关节驱动 ==========
auto link1 = std::make_shared<RigidBody>();
link1->mass = 2.0;
world->addBody(link1);

// 创建旋转副
auto joint1 = std::make_shared<RevoluteJoint>(
    base, link1, Vector3d::UnitZ(), Vector3d(0, 0, 0.5)
);
world->addJoint(joint1);

// 关节力矩控制
auto actuator1 = std::make_shared<JointActuator>(joint1);
actuator1->setControlMode(JointActuator::ControlMode::VELOCITY_CONTROL);
actuator1->setTargetVelocity(0.5);  // 0.5 rad/s
actuator1->setPDGains(100.0, 10.0);

// ========== 力控制：末端执行器 ==========
auto gripper = std::make_shared<RigidBody>();
gripper->mass = 0.5;
world->addBody(gripper);

// 外力控制
auto force_control = std::make_shared<ExternalForceForm>();
force_control->addBody(gripper);
world->addForm(force_control);

// 仿真循环
double t = 0.0, dt = 0.01;
for (int i = 0; i < 1000; ++i) {
    // 更新关节控制
    actuator1->computeControlOutput();
    
    // 更新外力（如夹取力）
    if (t > 2.0) {
        force_control->setForce(gripper, Vector3d(0, 0, -5.0));  // 向下5N
    }
    
    // 物理求解
    solver->step(*world, dt);
    
    t += dt;
}
```
