## 二、缺失组件（需要实现）

### 2.1 机械手/夹爪模型 ⭐⭐⭐ 高优先级

| 缺失组件 | 说明 | 复杂度 | 参考来源 |
|---------|------|-------|---------|
| **ParallelGripper 类** | 平行夹爪模型，包含左右两个手指 | 中 | ❌ 无参考 |
| **GripperFinger 类** | 夹爪手指（刚体 + 接触面） | 低 | ❌ 无参考 |
| **GripperBase 类** | 夹爪基座（连接到机械臂末端） | 低 | ❌ 无参考 |
| **夹爪关节驱动** | 通过PrismaticJoint驱动手指开合 | 中 | ⚠️ 可用现有Joint |
| **夹爪状态管理** | 开合宽度、夹取力、接触状态 | 中 | ❌ 无参考 |

**建议实现**:
```cpp
class ParallelGripper {
    RigidBody* leftFinger_;      // 左手指
    RigidBody* rightFinger_;     // 右手指
    RigidBody* base_;            // 基座
    PrismaticJoint* leftJoint_;  // 左关节
    PrismaticJoint* rightJoint_; // 右关节
    
    double maxAperture_;         // 最大开合宽度
    double currentForce_;        // 当前夹取力
    bool isGrasping_;            // 是否处于夹取状态
    
public:
    void setAperture(double width);      // 设置开合宽度
    void setGripForce(double force);     // 设置夹取力
    bool checkGrasp(const RigidBody* object); // 检测是否成功夹取
};
```

### 2.2 抓取力计算 ⭐⭐⭐ 高优先级

| 缺失组件 | 说明 | 复杂度 | 参考来源 |
|---------|------|-------|---------|
| **抓取约束模型** | 双向约束（非仅排斥） | 高 | ❌ 无参考 |
| **力/力矩控制** | 控制夹取力大小 | 中 | ❌ 无参考 |
| **抓取稳定性检测** | 判断物体是否稳定夹持 | 中 | ❌ 无参考 |
| **软接触模型** | 考虑手指变形的接触 | 高 | ❌ 无参考 |

**技术难点**:
- 现有 IPC 屏障: $b(d) = -\kappa(d-\hat{d})^2\ln(d/\hat{d})$ 仅提供排斥力
- 抓取需要**吸引力**或**双向约束**来保持物体
- 需要修改屏障函数或添加弹簧约束

**建议方案**:
```cpp
class GraspingConstraint : public Constraint {
    // 当物体在夹爪之间时，添加双向约束
    // 力 = -k * (d - d_target) - c * v_rel
};
```

### 2.3 开环控制系统 ⭐⭐⭐ 高优先级

| 缺失组件 | 说明 | 复杂度 | 参考来源 |
|---------|------|-------|---------|
| **KinematicBody 类** | 运动学刚体（不受力，按轨迹运动） | 中 | ❌ 无参考 |
| **Trajectory 基类** | 轨迹接口 | 低 | ⚠️ PolyFEM有类似概念 |
| **FixedTrajectory** | 固定位姿 | 低 | ❌ 无参考 |
| **LinearTrajectory** | 匀速直线运动 | 低 | ❌ 无参考 |
| **KeyframeTrajectory** | 关键帧插值（线性/样条） | 中 | ⚠️ PolyFEM有Bspline |
| **OpenLoopController 类** | 开环控制器 | 中 | ❌ 无参考 |

**已有设计方案**（在 `documents/2026.02.21/开环控制实现方案.md` 中）:
```cpp
class Trajectory {
public:
    virtual void getPose(double t, Vector3d& pos, Quaterniond& ori) const = 0;
    virtual void getVelocity(double t, Vector3d& vel, Vector3d& angVel) const = 0;
};

class KinematicBody : public RigidBody {
    std::shared_ptr<Trajectory> trajectory_;
public:
    void updateFromTrajectory(double t);  // 从轨迹更新位姿
};
```

**参考代码来源**:
- `temps/polyfem-main/polyfem-main/src/polyfem/utils/BSplineParametrization.cpp` - B样条参数化
- `temps/polyfem-main/polyfem-main/src/polyfem/utils/CubicHermiteSplineParametrization.hpp` - 三次Hermite样条

## 四、实施建议

### 4.1 最小可行实现（MVP）

如果只需要一个简单的夹取演示，按以下优先级实现：

```
Phase 1: 开环控制（已有设计方案）
    ├─ KinematicBody
    ├─ Trajectory
    └─ OpenLoopController
    
Phase 2: 平行夹爪
    ├─ ParallelGripper
    ├─ GripperFinger
    └─ 基于PrismaticJoint的驱动
    
Phase 3: 简单抓取
    ├─ 修改屏障函数或添加弹簧约束
    ├─ 基本的抓取力计算
    └─ 简单的抓取检测
```

### 4.2 完整实现

如果需要完整的机械手夹取系统：

```
Phase 1-3: 同上

Phase 4: 逆运动学
    ├─ IKSolver接口
    ├─ JacobianIK
    └─ CCD IK
    
Phase 5: 关节限制
    ├─ JointLimit
    └─ LimitConstraintForm
    
Phase 6: 机械臂模型
    ├─ SerialManipulator
    ├─ Link
    └─ EndEffector
    
Phase 7: 闭环控制
    ├─ PIDController
    └─ ForceController
    
Phase 8: 抓取评估
    ├─ GraspDetector
    └─ GraspQualityMetric
```
